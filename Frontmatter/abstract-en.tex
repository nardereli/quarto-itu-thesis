This section is reserved for the English summary. According to the ITU thesis
writing guide, Turkish theses require an extended English summary of 3-5 pages.
The summary should introduce the thesis topic, describe the methods used, and
state the main conclusions. References, figures, and tables should not be placed
in the summary.

This sample file is intentionally short so that the repository remains a compact
template. Replace it with the complete summary required for the final thesis.
